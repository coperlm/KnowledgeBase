# 生命与演化 (Life and Evolution)

这一部分关注生命系统的层级结构与演化机制：从分子与细胞，到个体与群落，再到医学与人体系统。重点是机制、证据与尺度转换。

## 这里关注什么
- 生命的共同机制：遗传信息、代谢网络、调控与稳态
- 层级结构：分子→细胞→组织→个体→群体/生态系统
- 演化视角：选择、漂变、约束、适应与权衡
- 方法与证据：实验、统计推断、组学数据与模型

## 本目录建议沉淀的内容
- 关键通路/机制的‘输入—输出—调控—例外’笔记（例如转录调控）
- 常用实验方法与局限（对照组、混杂因素、可重复性）
- 概念对照：表型/基因型、相关/因果、适应/副产物等

## 写作/学习提示（针对本主题）
- 记录每个结论的证据层级：体外/动物/人群、观察/干预
- 遇到争议：把‘机制不清’与‘证据不足’分开写
- 对复杂系统：先写变量、因果方向与潜在混杂，再写故事
