# 生物学的基础

基础生物学适合沉淀‘机制型笔记’：把结构、过程与调控关系写成可复用的因果链，而不是背事实。

## 这里关注什么
- 细胞与分子：膜、细胞器、蛋白结构功能、信号转导
- 遗传与表达：DNA/RNA、复制/转录/翻译、表观遗传与调控
- 代谢与能量：ATP、呼吸/光合、代谢通路与调控
- 实验方法：PCR/测序/电泳/显微与基本统计（按需）

## 本目录建议沉淀的内容
- 关键机制图：画出通路并标注调控点与常见实验读出
- 术语与缩写表（生物学最容易被缩写拖累）
- 常见实验设计套路：对照、随机、盲法、重复与效应量意识

## 写作/学习提示（针对本主题）
- 写任何通路：标注‘必需/充分’证据分别来自哪些实验
- 遇到新技术：记录它测量的究竟是‘什么变量’，以及分辨率/偏差
- 把“现象描述”与“机制解释”分段写，避免混写成故事
