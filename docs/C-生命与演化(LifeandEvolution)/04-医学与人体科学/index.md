# 医学与人体科学

这里用于整理人体系统与医学证据的框架：生理机制、疾病谱、诊疗证据等级与研究方法。避免把未经证实的健康建议当作结论。

## 这里关注什么
- 人体系统：神经、免疫、内分泌、循环、呼吸、消化等的机制概览
- 疾病机制：炎症、肿瘤、代谢、感染、退行性过程的共性逻辑
- 医学证据：RCT/队列/病例对照、系统综述、偏倚与混杂
- 诊断与测量：指标的敏感度/特异度、阈值与人群差异

## 本目录建议沉淀的内容
- 证据等级笔记：什么结论来自机制推断、什么结论来自临床试验
- 常见偏倚与混杂清单（尤其是健康科普最容易踩坑的部分）
- 你关注的疾病/主题的‘机制—证据—争议点’三段式摘要

## 写作/学习提示（针对本主题）
- 写任何‘有效/无效’判断时：同时记录研究设计、样本与结局指标
- 区分‘生理指标改善’与‘临床结局改善’，不要混写
- 对健康建议：优先记录权威指南结论与适用人群范围（不写操作性处方）

## 边界（避免写到别处）
- 具体诊疗步骤与处方级建议（不在此处沉淀，避免误用）
