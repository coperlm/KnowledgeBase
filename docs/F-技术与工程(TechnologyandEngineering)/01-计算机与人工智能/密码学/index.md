# 密码学科研

这里用于沉淀密码学科研向笔记：安全定义、威胁模型、构造与归约证明、实现注意点，以及论文阅读的结构化记录。

## 这里关注什么
- 威胁模型：被动/主动、选择密文/选择消息、适应性攻击等
- 安全定义：IND-CPA/CCA、EUF-CMA、模拟式/不可区分式等（按需）
- 构造范式：对称、公开密钥、签名、哈希、KDF、认证与密钥交换
- 高级主题：零知识证明、MPC、同态/函数加密、后量子、可组合安全（按兴趣）
- 实现与工程：侧信道、随机性、参数选择与协议边界（高层次）

## 本目录建议沉淀的内容
- 论文阅读卡：问题—模型—假设—构造—证明结构—对比—局限—开放问题
- 安全归约模板：Game hopping/Hybrid argument/模拟器构造的常见套路
- 原语对照表：同一目标的不同方案、假设强弱与性能权衡

## 写作/学习提示（针对本主题）
- 读论文先找：安全定义写在哪里？对手能力是什么？假设是否可接受？
- 写证明时：把每个 game 的差异点写成一句话，并标注用到的引理
- 对协议：明确参与方状态机与失败处理（尤其是认证与重放）

## 边界（避免写到别处）
- 具体攻击实施教程或非法用途（不在此处沉淀）
