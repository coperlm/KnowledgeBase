# 学习工具与技能树

这里用于沉淀“学习这件事本身”的工具链与方法论：从信息获取、阅读与笔记，到复习、输出与项目化训练。它更像你的个人操作系统（Personal OS）。

## 这里关注什么
- 信息输入：信息源筛选、RSS/订阅、收藏与归档策略
- 阅读与理解：速读/精读的切换、做笔记的粒度、如何提炼问题
- 记忆与复习：间隔重复（SRS）、卡片设计、错题/易错点管理
- 写作与表达：把知识变成可分享的文章、讲义、演示与代码
- 工具与环境：编辑器、终端、脚本、版本控制、自动化与模板
- 技能树：把“我想学什么”拆成可训练的子技能与评估标准

## 本目录建议沉淀的内容
- 工具链清单：你当前用的工具、配置要点、替代方案与选择理由
- 模板库：读书笔记模板、论文阅读卡、项目复盘模板、学习计划模板
- 方法对照：不同学习方法适用场景与代价（例如记忆 vs 理解 vs 产出）
- 常见坑位：例如“收藏癖”“工具癖”“不复盘”“只输入不输出”等的对策
- 训练设计：最小练习单元、反馈回路、如何衡量进步（可观察指标）

## 写作/学习提示（针对本主题）
- 任何工具/方法至少补齐三件事：适用场景、操作步骤（高层次）、验收标准
- 记录“你为什么改了工具/方法”：触发点是什么、效果如何、有没有副作用
- 把技能拆成“可练的动作”：例如写摘要、做推导、复现实验、讲给别人听
