# 边界与元知识 (The Frontier and Meta-Knowledge)

这一部分用于整理‘知识如何推进以及如何出错’：前沿趋势、未知问题、方法论与认知偏差。重点是证据标准与判断框架。

## 这里关注什么
- 前沿趋势：技术/社会/科学的可能走向与不确定性来源
- 元方法：研究方法论、可重复性、测量与模型偏差
- 未知与悖论：概念冲突、未解问题与开放问题的结构化记录
- 认知与信息环境：谣言、误导、激励扭曲与信息鉴别

## 本目录建议沉淀的内容
- 判断框架：证据等级、可证伪性、预测记录（forecasting log）
- 开放问题清单：问题定义—已知边界—关键障碍—可能突破口
- 信息卫生规则：来源分级、交叉验证、更新记录与撤回机制

## 写作/学习提示（针对本主题）
- 对任何前沿观点：写概率、依据、反证条件与更新时间（避免口号化）
- 对任何争议：把‘分歧点’精确到可检验的命题上
- 记录你改变观点的原因（证据出现/定义改变/模型修正）
