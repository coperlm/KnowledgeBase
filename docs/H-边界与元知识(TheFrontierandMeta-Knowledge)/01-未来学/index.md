# 未来学

未来学更像‘不确定性下的推理’：趋势不等于预测，关键在假设、驱动因素、情景与可验证的预警指标。

## 这里关注什么
- 驱动因素：技术、人口、制度、资源与文化的长期变量
- 情景分析：多条可能路径与关键分叉点
- 预测与校准：概率预测、基准率、预测赛与事后评估
- 预警指标：哪些信号意味着情景切换正在发生

## 本目录建议沉淀的内容
- 情景卡：假设—驱动—关键事件—可观察信号—反证条件
- 预测日志：时间戳、概率、依据、更新与复盘结果
- 方法论笔记：常见认知偏差与如何做校准（概念层）

## 写作/学习提示（针对本主题）
- 写趋势时：同时写 1) 反趋势力量 2) 约束条件 3) 失效路径
- 给出数字前：先写基准率与历史类比，再写差异点
- 定期复盘：预测对/错的原因（信息不足还是模型偏误）
