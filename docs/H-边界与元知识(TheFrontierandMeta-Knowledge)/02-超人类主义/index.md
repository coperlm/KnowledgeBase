# 超人类主义

超人类主义涉及技术、伦理与政治的交叉：增强的可能性、风险分布、治理机制与价值冲突。建议以‘论证结构+风险框架’记录。

## 这里关注什么
- 增强技术谱系（高层次）：生物技术、脑机接口、AI 辅助、义体与延寿等
- 伦理问题：公平、同意、身份、责任与不可逆后果
- 治理与制度：监管、标准、军民两用与国际协调难题
- 哲学基础：人是什么、何谓更好、价值冲突如何裁决

## 本目录建议沉淀的内容
- 议题地图：主张—理由—反驳—回应（钢人化）
- 风险表：技术风险/社会风险/治理失败风险与缓解手段（高层次）
- 概念澄清：增强 vs 治疗、个体利益 vs 公共利益等

## 写作/学习提示（针对本主题）
- 写任何立场：同时写‘在什么条件下我会改变观点’
- 对技术乐观/悲观论：分别列出最强支持论据与最强反对论据
- 优先记录可操作的治理问题，而不是空泛愿景
