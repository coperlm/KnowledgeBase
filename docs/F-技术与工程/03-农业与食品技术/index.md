# 农业与食品技术

农业与食品技术可以从‘生产系统—过程控制—质量安全’组织：原料、工艺、微生物/化学过程与质量检测相互耦合。

## 这里关注什么
- 农业生产：土壤、水分、营养、病虫害与环境控制（概念层）
- 食品加工：工艺流程、热处理、发酵、保鲜与包装（概念层）
- 质量与安全：风险点、检测指标、追溯与标准体系（高层次）
- 供应链：冷链、损耗、成本与可持续性（概念层）

## 本目录建议沉淀的内容
- 过程参数清单：温度/时间/湿度/菌种等与目标质量的关系（概念层）
- 风险点梳理：污染源、交叉污染、过敏原、储运条件（高层次）
- 你关心的产品/工艺的案例笔记：目标—约束—验证指标—失败模式

## 写作/学习提示（针对本主题）
- 记录任何工艺时：写清关键控制点（CCP）与监测指标（高层次）
- 区分‘感官质量’与‘安全合规’，分别记录指标与验证方式
- 对改进方案：记录成本、可操作性与规模化风险
