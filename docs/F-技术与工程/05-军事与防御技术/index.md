# 军事与防御技术

这里仅做高层次的防务技术与安全治理笔记：组织、体系、通信与信息系统、安全边界、历史演进与伦理风险。避免任何可直接用于制造/改装武器的操作性内容。

## 这里关注什么
- 体系与组织：指挥控制、情报、通信、后勤与协同（概念层）
- 信息安全：通信安全、抗干扰、网络防护与风险管理（高层次）
- 技术史：关键技术如何改变作战与组织形态（以历史理解为主）
- 伦理与治理：军民两用、出口管制、误用风险与责任边界

## 本目录建议沉淀的内容
- 概念框架：威胁建模、风险评估、冗余与韧性（resilience）笔记
- 案例复盘：事件—技术因素—组织因素—制度教训（不涉及操作细节）
- 术语表：C4ISR、电子战等概念的自用解释与边界

## 写作/学习提示（针对本主题）
- 写任何技术时：优先写‘防护/治理/韧性’而不是‘攻击/破坏’
- 对军民两用：记录潜在误用路径与缓解措施（流程与制度层面）
- 对历史案例：把技术因素与决策因素分开写，再讨论结论

## 边界（避免写到别处）
- 武器制造、改装、使用的教程式内容（不写）
