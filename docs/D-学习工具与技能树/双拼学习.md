## 环境搭建

我使用的是`小鹤双拼`，下面是配置方式

你问我为什么选择小鹤？因为用的人最多~

### windows 下，使用默认输入法

我用的是默认的微软输入法

首先打开命令行

![win1](双拼学习/win1.png)

输入这一串内容

```
reg add HKCU\Software\Microsoft\InputMethod\Settings\CHS /v UserDefinedDoublePinyinScheme0 /t REG_SZ /d "小鹤双拼*2*^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt" /f
```

然后回车即可

![win2](双拼学习/win2.png)

打开`设置-时间和语言-语言和区域-微软拼音输入法-常规-拼音设置`

直接改为双拼即可

然后下面选择`小鹤双拼`

![win3](双拼学习/win3.png)

即可享用双拼
