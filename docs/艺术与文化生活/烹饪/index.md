# 烹饪

烹饪目录建议沉淀‘可复现的做饭系统’：基础技法、风味构建、火候与时间控制、食材替代与安全卫生。

## 这里关注什么
- 基础技法：切配、煎炒炖烤、勾芡、乳化与收汁
- 风味结构：咸甜酸苦鲜、香料、脂香与层次构建
- 过程控制：火候、温度、时间与水分管理
- 食材与替代：常见替代方案、季节性与成本
- 安全卫生：生熟分离、储存、过敏原与清洁（高层次）

## 本目录建议沉淀的内容
- 菜谱卡：目标口感/风味→关键步骤→关键温度/时间→验收标准→常见失败原因
- 基础酱汁/底味库：能复用到多道菜的“母配方”
- 食材处理备忘：解冻、腌制、去腥、保水等（强调安全与可复现）

## 写作/学习提示（针对本主题）
- 写菜谱时：用‘验收标准’描述成熟度（颜色、质地、出汁情况）
- 做改动时：一次只改一个变量（盐、糖、火、时间、油量）便于学习
- 记录替代方案：没有某食材/设备时怎么做出接近效果
