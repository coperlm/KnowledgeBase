# 系统与信息论

系统与信息论提供一套‘用结构与信息度量理解世界’的方法：系统如何演化、信息如何产生/传递/压缩、复杂性如何刻画。

## 这里关注什么
- 系统观点：状态、输入输出、反馈、稳定性、控制直觉
- 信息度量：熵、互信息、KL 散度的直观解释与常用不等式
- 编码与通信：信道、纠错码、容量与可靠性（按需）
- 复杂系统：涌现、网络结构、相变式行为、可观测性/可控性（按需）

## 本目录建议沉淀的内容
- 常用信息量公式与适用条件（写清楚随机变量与独立性假设）
- 你在论文/工程里遇到的“系统建模套路”：状态空间/图模型/马尔可夫模型
- 典型例子：压缩、分类、假设检验、最大熵原理的最小推导

## 写作/学习提示（针对本主题）
- 遇到‘信息’相关说法：问清楚信息源、编码方式、度量标准分别是什么
- 画出系统框图：哪些是状态、哪些是观测、噪声从哪里进入
- 把抽象量（熵/互信息）对应到你能解释的现象/直觉上
