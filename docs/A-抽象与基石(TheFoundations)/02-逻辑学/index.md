# 逻辑学

逻辑学是把‘推理’变成对象来研究：哪些推理规则可靠？形式系统能表达什么？证明与模型之间如何对应？

## 这里关注什么
- 命题逻辑与谓词逻辑：语法/语义、可满足性、推导系统
- 证明系统：自然演绎、序列演算、分辨率（按需选一种深入）
- 元理论：可靠性/完备性、紧致性、可判定性与不可判定性直觉
- 模型与结构：解释、模型、理论、同构（面向理解即可）
- 与计算的连接：SAT/SMT、自动定理证明、类型系统与 Curry–Howard（按需）

## 本目录建议沉淀的内容
- 常用推理规则速查：等价变形、量词推进、Skolem 化（若用到）
- 经典反例库：看似合理但不成立的推理形式（帮助查错）
- 你常用的‘形式化模板’：如何把自然语言命题翻译为公式

## 写作/学习提示（针对本主题）
- 写证明时：显式标注每一步使用的规则（能暴露‘偷换概念’）
- 遇到悖论/矛盾：先区分‘语义悖论’与‘集合论悖论’再记录
- 对一个系统：回答‘能表达什么/不能表达什么/代价是什么’
