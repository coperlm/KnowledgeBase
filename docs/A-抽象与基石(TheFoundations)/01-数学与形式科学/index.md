# 数学与形式科学

数学与形式科学更像‘语言与工具箱’：它提供可验证的结构、推理与抽象能力，为后续所有学科提供共同底座。

## 这里关注什么
- 集合与结构：集合、关系、函数、等价类、偏序
- 代数与离散：群/环/域、图论、组合、生成函数
- 分析与连续：极限、微积分、测度、泛函分析的直觉
- 概率与统计：随机变量、期望方差、集中不等式、估计与检验
- 计算与形式化：复杂度、自动机、可计算性、类型论/范畴直觉（按需）

## 本目录建议沉淀的内容
- 你自己能‘复述并使用’的核心定理清单（每条带 1 个最小应用）
- 常见证明与构造例子：反例、极值构造、对称性利用
- 易混概念对照表：必要/充分、几乎处处/处处、收敛类型等

## 写作/学习提示（针对本主题）
- 每学一个概念：写‘我什么时候会用到它’以及‘不用它会卡在哪’
- 对每条定理：记录一个‘典型题型’与一个‘陷阱题型’
- 把你常用的推导步骤写成可复用的“推导脚本”（例如换元、界估计）
