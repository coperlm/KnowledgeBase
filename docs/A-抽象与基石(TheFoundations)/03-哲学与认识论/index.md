# 哲学与认识论

这里用来整理‘我们如何知道’以及‘概念如何成立’：定义的边界、解释的层级、证据与信念更新、理论选择标准。

## 这里关注什么
- 认识论：知识的条件、证成、怀疑论、贝叶斯更新的哲学含义
- 科学哲学：可证伪性、解释与预测、理论变迁与范式
- 形而上学（适量）：本体论承诺、因果、同一性与反事实
- 语言哲学（与 E 目录交叉）：意义、指称、语境与推理

## 本目录建议沉淀的内容
- 常用概念的‘定义家族’：不同学派/语境下的定义与差异
- 论证结构笔记：主张—理由—反驳—回应（避免只摘抄观点）
- 方法论清单：如何评价证据强度、如何处理模型假设与价值判断

## 写作/学习提示（针对本主题）
- 看到一个观点：写出它的隐含前提与可反驳点（而不是只记录结论）
- 对争论问题：列出‘分歧来自定义/事实/价值/推理’中的哪一类
- 把你常用的‘世界观假设’显式化，便于以后修正
