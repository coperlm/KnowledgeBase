## 背单词

```
我高中的时候尝试使用百词斩背单词，那时背诵高考词汇，是系统给多个图片，用户选择匹配的图片对应的释义，故我对图片的记忆很深刻，经常看到一个单词先想到它的图片然后再想到释义，那时成绩也有提升；现在我需要准备考研，打开百词斩下载考研词汇，发现已经今非昔比了，我也用不习惯（也是因为词书不同，形式不同了）；于是乎我转向现在更为热门的不背单词app，我在高强度学习了几天之后，发现了障碍：不背单词是没有图片，也有例句和读音，相对简洁很多，但是需要更多“用脑子”去记忆单词，我如果不带脑子去刻意记住单词，可能这一组单词一开始选择“不认识”，然后下一轮依旧“不认识”进而死循环，但是我的注意力又是有限的，我也在尝试很多方法，希望更多更快的背单词，请你为我具体的出谋划策，我已经决定继续使用不背单词app了，后续专注于如何高效背单词即可；不必和我提及一些老生常谈的方法，请你认真思考后再回答我

不要引入额外负担，这样会大幅加大执行难度！不要引入额外负担，这样会大幅加大执行难度！不要引入额外负担，这样会大幅加大执行难度！如果直接去想图片，我需要额外调用大脑的想象去构造这个场景，可能还不如直接记忆；如果抄写一遍单词，会引入额外的负担，并且并不一定能记住（我此前有过抄写单词，结果白抄，一点记不住）；我不理解你说的“用极度夸张的语气念一遍”，这样感觉有点“无病呻吟”且达不到记忆效果；关于“不要背 oblige，去背它高亮给你的短语，比如 be obliged to do（被迫做某事）。把生词绑架在它最常出现的几个熟词旁边”，我不觉得这是“生词绑架在它最常出现的几个熟词旁边”，而是额外需要记忆的固定搭配；词根词缀记忆法也是老生常谈了，但是实际操作下并不容易

我实践了一下，发现我背不过单词的根本原因还是急于求成；我分配给每日的工作量很大，故不得不急迫一点，这就导致了本末倒置，但是因为实际上并没有背过那部分单词，导致每日复习量骤增，故我觉得我有必要更改策略。我此前经常手指无意识地点击”不认识“（实际上也确实不认识）然后瞟一眼释义就下一个（不到半秒），然后这样刷过去很多词汇，最终也没有背过这些词汇；我决定后续背单词的时候，如果的确有”不认识“的单词，至少要停留直到例句读完才能下一个，这样才能”慢即快“。你还有进一步的观点看法吗

还有一个问题，不背单词复习模式下每组词的最终阶段，是”看中文，问你认不认识英文”，这一阶段的形式此前阶段都没有，也不能关闭。我想知道这种形式是否有必要去学习，我觉得考研英语核心还是认识单词然后去做阅读，作文什么的需要的词汇相对较少（并且我把拼写模式关闭了，计划第二轮再开拼写），故感觉作用不大，我该如何对待这部分呢
```
