# 视觉艺术

视觉艺术可以围绕‘形式语言’整理：构图、色彩、材质、图像符号如何在不同媒介中产生感受与意义。

## 这里关注什么
- 形式元素：构图、透视、光影、色彩、质感、节奏与重复
- 媒介与工艺：绘画/摄影/设计/影像的约束与可能性
- 图像叙事：符号、隐喻、视觉修辞与文化语境
- 观看与心理：注意力、对比、错觉与审美判断（概念层）

## 本目录建议沉淀的内容
- 作品分析卡：画面元素清单→组织方式→情绪效果→意义解读→证据
- 术语表：你常用的构图/色彩/版式术语与示例
- 参考库：你喜欢的作品与可借鉴的具体手法（但不抄）

## 写作/学习提示（针对本主题）
- 分析一幅作品：先描述可见事实，再给出解释（避免先入为主）
- 记录你为什么被打动：是形式、叙事、还是文化联想
- 对设计类作品：写清目标受众与信息层级
