# 教育与知识传播

教育与知识传播的重点是‘如何让知识可迁移’：结构化表达、练习设计、反馈机制与评估，而不是堆资料链接。

## 这里关注什么
- 教学设计：学习目标、前置知识、分层练习与迁移
- 解释与表达：类比、例子、反例、可视化与叙事结构
- 评估与反馈：测验、作业、rubric、间隔重复与形成性评价
- 知识管理：笔记系统、索引、检索与版本迭代

## 本目录建议沉淀的内容
- 课程/主题大纲：目标—概念图—练习序列—常见误区
- 讲解模板：定义→最小例子→反例→常见坑→练习→总结
- 传播材料：讲稿、幻灯、练习题与答案解析（按需）

## 写作/学习提示（针对本主题）
- 写任何知识点：补一个最小练习与一个常见错误示例
- 把‘我懂了’转成‘别人能做对’：写验收标准与反馈方式
- 对内容迭代：记录学习者在哪一步卡住、你如何改讲法
