# 工艺与造物

工艺与造物用于沉淀‘可重复的手艺’：材料、工具、过程控制、质量标准与复盘。重点是把感觉变成可描述的参数与检验标准。

## 这里关注什么
- 材料与工具：性质、选择理由、替代方案与维护
- 过程控制：步骤、关键参数、容错范围与常见失败原因
- 质量与审美：验收标准、边界条件与改进方向
- 安全：操作安全、食品/化学品安全与环境卫生（高层次）

## 本目录建议沉淀的内容
- 流程卡：输入材料→步骤→关键参数→验收标准→失败排查
- 配方/工艺变体对照：不同参数带来的效果差异
- 工具维护与耗材管理：保养周期、替换条件与经验

## 写作/学习提示（针对本主题）
- 每次实践：记录 1~2 个你改变的变量与结果（做实验而不是凭感觉）
- 失败也要记：失败原因往往比成功更可复用
- 把‘口感/手感/观感’翻译成可度量或可检验的标准
