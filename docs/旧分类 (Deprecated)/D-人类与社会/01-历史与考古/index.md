# 历史与考古

历史与考古笔记建议围绕‘证据与解释’：同一事件可有多种解释，关键在史料来源、偏差与可反驳性。

## 这里关注什么
- 史料类型：一手/二手、文献/器物/遗址/口述与各自偏差
- 时间线与结构：事件史与结构史（经济、技术、制度、文化）
- 考古方法：分期、测年、遗址解读与不确定性
- 比较与反事实：不同地区/制度对比，解释变量与路径依赖

## 本目录建议沉淀的内容
- 事件条目模板：时间—地点—参与者—证据—争议点—不同解释
- 史料批判笔记：作者立场、写作目的、传播链与缺失信息
- 主题综述：例如国家形成、城市化、技术扩散、贸易网络等

## 写作/学习提示（针对本主题）
- 对一个观点：至少记录一个支持证据与一个反对证据（或缺证据）
- 把‘发生了什么’与‘为什么发生’分段写，避免叙述替代理解
- 对考古结论：写清推断链条中的不确定性来源
