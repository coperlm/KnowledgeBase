# 信仰与神学

这里用于以‘理解与比较’为目标整理信仰传统：教义、仪式、伦理与共同体结构，以及解释传统如何演化。

## 这里关注什么
- 核心概念：神/超越、启示、救赎、因果与意义（不同传统差异）
- 经典文本与诠释：文本、注疏、教义形成与分歧点
- 仪式与共同体：实践如何塑造身份与道德规范
- 宗教与社会：政治、文化、艺术与社会组织的互动

## 本目录建议沉淀的内容
- 传统对照表：同一问题在不同传统中的回答与前提差异
- 概念澄清笔记：哪些是神学命题、哪些是历史/社会学描述
- 经典争议点梳理：分歧来自文本解释、权威结构还是价值选择

## 写作/学习提示（针对本主题）
- 写任何比较：先写各传统的自我理解，再写外部分析框架
- 区分信仰陈述与可经验检验命题，避免类别错误
- 对教义：记录它解决的‘存在性问题’是什么，而不仅是条款
