# 表演与听觉艺术

表演与听觉艺术更适合用‘结构—动机—效果’记录：音乐/表演如何在时间轴上组织张力、情绪与意义。

## 这里关注什么
- 音乐结构：动机、主题发展、和声/节奏/配器（按需）
- 表演结构：场面调度、节奏、角色关系、舞台/镜头语言（按需）
- 听觉心理：期待、重复与变化、注意力与记忆（概念层）
- 跨媒介：影视配乐、声音设计、戏剧与叙事的协同

## 本目录建议沉淀的内容
- 作品时间线笔记：关键段落—发生了什么—造成什么效果—为什么
- 术语/技巧表：你常遇到的节奏型、和声走向、表演调度手法
- 参考片段库：你想反复学习的片段与其结构分析

## 写作/学习提示（针对本主题）
- 对一个片段：写‘前后对比’（变化点）而不是只写好听/好看
- 把主观感受翻译成可描述的结构（节奏密度、音区、张力点）
- 区分‘作品结构’与‘演绎版本’的差异
