# 能源与环境技术

能源与环境技术的主线是‘能量流与物质流’：发电/储能/输配/用能效率，以及污染物产生、迁移与治理的工程闭环。

## 这里关注什么
- 能源系统：发电、储能、输配电、负荷与调度（概念层）
- 可再生：风光、水电、生物质与并网问题（概念层）
- 环境工程：水/气/固废处理与监测（概念层）
- 碳与气候：排放核算、减排路径、生命周期分析（概念层）

## 本目录建议沉淀的内容
- 系统边界笔记：LCA 边界、排放口径、效率定义与常见误读
- 技术对照：不同储能/治理方案的优势、约束与适用场景（高层次）
- 指标库：效率、成本、可靠性、环境影响的评价框架

## 写作/学习提示（针对本主题）
- 写任何‘更环保’结论：同时写系统边界与转移污染风险
- 对能源方案：写清峰谷、可调度性与可靠性约束，而不只看均值
- 对治理技术：记录输入水质/气质/负荷范围与失效条件
