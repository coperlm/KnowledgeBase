# 腹式呼吸

腹式呼吸大家都会，难的是如何一直使用腹式呼吸

我现在（260531）还没学会，还在练习

大概内容就是 首要前置条件是需要坐直

1. 坐直大家都会，但是最多几分钟，除非是训练过，不然坚持几十分钟都很少见，但是并不需要坐得太直
2. 坐骨（摸摸自己的屁股，里面每边有两块硬硬的骨头）直接接触椅子（凳子也行），不要使用坐骨后面那一堆软软的肉肉，不然不仅坐不直 还会腰间盘突出不拉不拉（恐吓）
3. 坐骨坐直之后，腰部还会自然弯曲，可以在腰部（和肚脐差不多的高度，但是是后背）放一个圆柱形的软垫，就像有人在后面推着自己坐直一样
4. 屁股和腰都直了，脖子也不能弯曲，大抵想象自己的耳朵和肩膀同线即可，具体可以感受一下，真伸脖子的话感觉还是挺明显的
5. 之后就勤加练习腹式呼吸，绝大多数人都失败在了坐直，还有就是不要翘二郎腿喵
