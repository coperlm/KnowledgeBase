# 救赎论

救赎论是一般指的就是基督教神学对救赎的观点。[维基百科][https://zh.wikipedia.org/zh-cn/%E5%9F%BA%E7%9D%A3%E6%95%99%E6%95%91%E8%B4%96%E8%AB%96]上的内容有些过于晦涩了，故这里通俗介绍一下。

首先列举前提：

1. 人是有原罪的（亚当夏娃吃苹果）
2. 人无法靠自己赎完罪（即便是世界首富，也不足以还清原罪）
3. 赎完罪的人才能上天堂，不然就下地狱去吧~
4. 做好事能赎罪

很容易想到，赎不完罪那为啥还要赎罪——我要干无穷多的好事才能上天堂，还不能干一件坏事，那我肯定上不了天堂了；天堂让耶稣一个人待在那里好了，我就开摆了~
