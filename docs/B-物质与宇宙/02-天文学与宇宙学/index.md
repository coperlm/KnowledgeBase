# 天文学与宇宙学

天文学与宇宙学的核心是‘观测约束下的推断’：从光谱/视差/红移等观测量反推出物理过程与宇宙参数。

## 这里关注什么
- 观测基础：望远镜/探测器、噪声、选择效应与校准
- 恒星与星系：形成、演化、金属丰度、反馈过程
- 宇宙学：宇宙学原理、膨胀史、CMB、结构形成
- 暗物质/暗能量：证据链、替代理论与观测区分点

## 本目录建议沉淀的内容
- 常用观测量与物理量的映射表：红移—距离—时间、光谱—成分—温度
- 典型证据链梳理：每个结论依赖哪些观测与哪些模型假设
- 常见系统误差/偏差清单（选择效应、尘埃消光等）

## 写作/学习提示（针对本主题）
- 写任何‘宇宙学结论’时：同时写出模型假设与可替代解释
- 看到相关图（H-R 图、功率谱等）：记录坐标含义与可读出的信息
- 把‘观测事实’与‘理论解释’分段写，避免混在一起
