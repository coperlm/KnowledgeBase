# 个人发展与生产力

个人发展与生产力建议以‘系统设计’而非意志力叙事：目标拆解、环境设计、反馈回路与复盘机制。

## 这里关注什么
- 目标与规划：长期方向、阶段里程碑、优先级与取舍
- 时间与注意力：任务切换成本、深度工作、信息饮食
- 习惯与行为：触发—行为—奖励回路、环境设计、阻力管理
- 工具与流程：任务系统、笔记系统、自动化与复盘

## 本目录建议沉淀的内容
- 个人系统说明书：你的工作流、工具链、规则与例外
- 复盘模板：目标—事实—原因—改进—下一次实验
- 清单：每日/每周例行、学习/输出配比、信息源白名单

## 写作/学习提示（针对本主题）
- 每次调整习惯：写一个可测试的实验周期与指标（两周就够）
- 记录能量管理：睡眠/运动/压力与产出之间的关系（事实为主）
- 对任何工具：写‘它替你省的步骤是什么’，否则容易工具控
