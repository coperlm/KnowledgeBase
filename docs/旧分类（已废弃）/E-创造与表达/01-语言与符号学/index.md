# 语言与符号学

语言与符号学关注‘意义如何被编码与理解’：符号系统的结构、语境对意义的影响，以及叙事/修辞如何改变认知。

## 这里关注什么
- 语言结构：语音/词法/句法/语义/语用的层级
- 符号学：能指/所指、符码、互文与文化语境
- 语用与对话：含义、预设、会话含义、礼貌策略
- 文本与叙事：信息结构、主题推进、修辞与论证

## 本目录建议沉淀的内容
- 概念对照：语义 vs 语用、指称 vs 描述、字面 vs 隐喻等
- 分析工具：如何做句法/语篇切分，如何标注信息结构
- 案例库：你在阅读/写作中见到的高频修辞与其效果

## 写作/学习提示（针对本主题）
- 分析一句话：先写字面语义，再写语境下的含义与预设
- 写修辞时：记录它改变的是注意力、情绪、还是推理路径
- 对符号系统：问清规则、歧义来源与解释共同体
