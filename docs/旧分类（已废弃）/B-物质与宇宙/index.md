# 物质与宇宙 (The Physical Universe)

这一部分关注自然界的可测量规律：从微观相互作用到宏观结构，从物质性质到宇宙演化。目标是把‘模型—实验/观测—可检验预测’的链条整理清楚。

## 这里关注什么
- 基本物理图景：守恒律、场与相互作用、统计规律
- 物质结构：原子/分子/晶体到材料性能
- 地球系统：地质—大气—海洋的耦合与长期演化
- 宇宙结构：恒星演化、星系尺度结构、宇宙学参数

## 本目录建议沉淀的内容
- 常用模型与近似条件：何时可忽略哪些项（这是最值钱的部分）
- 关键量纲/尺度分析笔记：把公式变成直觉
- 经典实验/观测的‘输入—输出—结论—局限’摘要

## 写作/学习提示（针对本主题）
- 每个公式：写出量纲检查与适用尺度（微观/宏观、低速/高速等）
- 每个现象：记录最小可解释模型与最先失败的情形
- 把你常用的数值常量与量级估计集中管理
