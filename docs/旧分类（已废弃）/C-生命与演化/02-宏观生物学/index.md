# 宏观生物学

宏观生物学关注形态、行为与生理如何在环境与演化压力下形成，并如何用比较与实验方法解释差异。

## 这里关注什么
- 形态学与功能：结构如何服务功能、尺度与力学约束
- 生理学：体温、循环、呼吸、内分泌与稳态调控（按需）
- 行为学：学习、决策、社会行为与适应性解释
- 比较方法：谱系、同源/趋同、性状演化与统计推断

## 本目录建议沉淀的内容
- 典型适应案例卡片：环境→压力→性状→证据→替代解释
- 比较方法备忘：如何避免把‘谱系相关’误当‘独立样本’
- 你常用的生理量与量级估计（能量消耗、代谢率等）

## 写作/学习提示（针对本主题）
- 写适应解释时：同时写一个‘非适应’解释（副产物/约束）
- 记录关键数据：样本量、效应量、置信区间/后验，而不只看显著性
- 把“机制层面”与“功能层面”分开写再汇总
