# 计算机与人工智能

这里用于沉淀计算机科学与 AI 的体系化笔记：从计算模型与算法，到系统工程与数据驱动方法。建议按‘抽象层’组织，避免零散堆叠。

## 这里关注什么
- 计算基础：数据结构、算法、复杂度与计算模型
- 系统：操作系统、网络、分布式、存储与一致性直觉
- 工程：软件架构、测试、可观测性、性能与安全（高层次）
- AI：机器学习基本范式、训练/泛化、评估与数据问题

## 本目录建议沉淀的内容
- 概念图谱：同一概念在不同层（算法/系统/工程）中的对应关系
- 常见 trade-off 清单：一致性 vs 可用性、延迟 vs 吞吐、精度 vs 速度等
- 你常用的评估指标与基准设置（避免只看单一指标）

## 写作/学习提示（针对本主题）
- 遇到新技术：先写它解决的瓶颈是什么，再写它引入的新约束
- 对任何论文/项目：写清实验设置与可复现要素（数据/代码/参数）
- 把失败经验也记录下来：什么条件下会崩、怎么监控与回滚
