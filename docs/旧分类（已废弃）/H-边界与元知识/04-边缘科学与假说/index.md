# 边缘科学与假说

这个目录的价值在于‘评估框架’，而不是追热点：边缘假说常混合真实难题与不可靠叙事。这里应沉淀可证伪性、证据链与方法论。

## 这里关注什么
- 可证伪性：假说如何被证明为错？需要什么观测/实验？
- 证据链：数据来源、测量误差、统计处理与替代解释
- 研究规范：可重复性、预注册、数据开放与同行评议的作用
- 心理与传播：动机、选择性报道、阴谋叙事与误导机制

## 本目录建议沉淀的内容
- 假说评估卡：主张—预测—可反驳条件—现有证据—替代解释—结论等级
- 常见谬误清单：相关即因果、样本偏差、p-hacking、幸存者偏差等
- 来源白名单/灰名单：哪些渠道通常可靠、哪些只作追踪不作证据

## 写作/学习提示（针对本主题）
- 写任何边缘主张：必须写出可检验预测与反证条件，否则不记录为‘知识’
- 将‘观察到的现象’与‘解释框架’分开写，避免概念偷换
- 如果只看到二手传播：先追到原始来源再下结论
