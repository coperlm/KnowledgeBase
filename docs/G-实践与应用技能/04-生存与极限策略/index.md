# 生存与极限策略

这里更适合做‘风险意识与应急体系’笔记：评估风险、准备资源、建立通讯与行动预案。避免提供可能造成危险的具体操作教程。

## 这里关注什么
- 风险评估：威胁类型、概率、影响、脆弱点与缓解策略
- 准备体系：物资、信息、通讯、路线与备份方案（高层次）
- 心理与团队：压力下决策、协同与沟通协议（概念层）
- 安全与合规：遵守当地法律、尊重救援体系与公共安全

## 本目录建议沉淀的内容
- 应急预案模板：触发条件—目标—行动步骤（高层次）—联络方式—复盘
- 物资与信息清单：按场景分类（停电/断网/出行受阻等，高层次）
- 经验复盘：事件—当时信息—决策—结果—下次改进（不写危险细节）

## 写作/学习提示（针对本主题）
- 写任何策略：优先写‘降低风险’而不是‘冒险技巧’
- 对预案：写最坏情况与撤退条件（何时停止行动）
- 把沟通协议写清楚：谁负责什么、多久汇报一次、失联怎么办

## 边界（避免写到别处）
- 可能导致人身伤害的具体操作教程（不写）
