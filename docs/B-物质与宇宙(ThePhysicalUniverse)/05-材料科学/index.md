# 材料科学

材料科学的主线是‘结构—工艺—性能—失效’：微观结构如何由工艺形成，如何决定宏观性能，以及在什么机制下失效。

## 这里关注什么
- 材料结构：晶体缺陷、相结构、晶粒/织构、非晶/聚合物结构
- 相图与相变：热处理、析出、扩散与动力学限制
- 性能：力学、电学、热学、光学、磁学与耦合效应
- 表征：显微/谱学/散射方法能回答的具体问题
- 失效分析：疲劳、腐蚀、蠕变、断裂韧性与断口学直觉

## 本目录建议沉淀的内容
- 常见材料体系的‘工艺—显微—性能’对照笔记（例如钢、铝合金、半导体）
- 常用性能指标与测试方法速查（以及容易误读的点）
- 你遇到的材料失效案例库：现象—机理—证据—改进方案（高层次）

## 写作/学习提示（针对本主题）
- 写材料结论时：明确样品制备与历史（工艺）信息，这是性能解释的前提
- 对任何表征图：写‘可支持的结论’与‘不能推出的结论’
- 把材料问题拆成：需求→约束→候选体系→验证指标→失效模式
