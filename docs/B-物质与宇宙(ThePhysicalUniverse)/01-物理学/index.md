# 物理学

物理学笔记建议围绕‘建模与近似’组织：同一套方程在不同尺度/边界条件下如何简化、对应哪些可观测量。

## 这里关注什么
- 经典力学：拉格朗日/哈密顿视角、对称性与守恒
- 电磁学：场方程、边界条件、波与辐射
- 量子力学：态与算符、测量、公设、近似方法
- 统计物理：系综、熵、涨落、相变直觉
- 相对论：时空几何、四矢量、极限情况

## 本目录建议沉淀的内容
- 常见近似：小振幅/弱耦合/低能有效理论等（写清适用范围）
- 典型题型：散射、简谐振子、两能级系统、黑体辐射等总结
- 你用过的推导链：从对称性到守恒、从哈密顿量到演化

## 写作/学习提示（针对本主题）
- 把每个模型的‘自由度’写清楚：你究竟在跟踪哪些量
- 遇到悖论式直觉冲突：先标注参考系/测量过程，再讨论结论
- 对每章：写一个‘能算出什么量、怎么和实验对应’的清单
