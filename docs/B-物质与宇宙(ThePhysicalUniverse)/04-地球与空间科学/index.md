# 地球与空间科学

地球与空间科学适合沉淀‘多尺度耦合系统’的理解：地质构造、气候系统、海洋循环与空间环境共同塑造可观测现象。

## 这里关注什么
- 地质与构造：板块构造、地震/火山、地层与地貌
- 大气与气候：辐射收支、环流、极端天气与气候变化证据
- 海洋与水循环：热盐环流、海气耦合、流域过程
- 空间环境：太阳风、磁层、电离层与空间天气（按需）
- 遥感与数据：反演、分辨率、误差来源（按需）

## 本目录建议沉淀的内容
- 常用时间尺度/空间尺度对照（天气—气候—地质尺度）
- 典型观测指标与含义：同位素、冰芯、卫星产品等
- 你关心地区/现象的案例档案（事件—数据—解释—争议点）

## 写作/学习提示（针对本主题）
- 写任何结论时：标注它属于‘观测事实/反演结果/模型预测’哪一类
- 对遥感产品：记录反演假设与适用场景，避免拿错数据解释
- 对复杂系统：先画能量/质量守恒的闭合关系再深入细节
