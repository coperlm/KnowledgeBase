# 化学

化学可以按‘结构—能量—动力学—表征’来组织：分子结构决定能量景观，动力学决定过程，表征把模型和实验连接起来。

## 这里关注什么
- 化学键与结构：轨道/杂化、分子构型、晶体结构
- 热力学：自由能、相平衡、溶液与活度（按需）
- 动力学：反应机理、速率方程、催化与扩散限制
- 有机/无机/物化的典型反应与规律（按你的兴趣侧重）
- 表征手段：光谱、色谱、质谱、XRD 等‘能说明什么/不能说明什么’

## 本目录建议沉淀的内容
- 常见反应类型与机理模板（箭推要点、决定速率步骤）
- 常见实验表征的判读笔记（谱峰对应什么结构信息）
- 你常用的能量与速率直觉：什么时候热力学可行但动力学不行

## 写作/学习提示（针对本主题）
- 每个机理：写一个‘关键实验如何区分机理’的想法
- 每个谱图：写清楚样品前处理与潜在干扰因素
- 不要只记结论：把‘为什么这样稳定/为什么这样反应’写出来
