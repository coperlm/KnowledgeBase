# 生态与演化

生态与演化适合沉淀‘动态过程’：种群如何变化、相互作用如何塑造群落、演化如何与生态过程耦合。

## 这里关注什么
- 种群动力学：增长、承载量、竞争/捕食/互惠模型直觉
- 群落与网络：食物网、稳定性、多样性与功能
- 演化机制：选择、漂变、突变、基因流与适应景观
- 宏观演化：谱系、多样化、灭绝事件与环境驱动

## 本目录建议沉淀的内容
- 常用模型对照：Lotka–Volterra、SIR、适应动力学等（按需）
- 参数意义与可测量性笔记：哪些参数可由数据估计，哪些只是抽象
- 经典案例：入侵物种、共同进化、性选择等的证据链摘要

## 写作/学习提示（针对本主题）
- 看到任何模型：写‘它忽略了什么’以及忽略后会在哪些情况下失败
- 对相关研究：记录尺度（时间/空间）与抽样方法，这是解释差异的关键
- 把“相关性结论”与“因果机制”分开，避免过度解释
