# 文学

文学笔记建议更像‘结构化阅读记录’：主题与人物只是起点，更重要的是叙事结构、语言策略与文本如何制造意义。

## 这里关注什么
- 体裁与结构：叙事视角、时间结构、可靠/不可靠叙述
- 主题与母题：反复出现的意象、隐喻与价值冲突
- 语言与风格：句式、节奏、意象、修辞与语域
- 文本与语境：历史语境、作者传统、互文关系

## 本目录建议沉淀的内容
- 读书卡：情节梗概（极短）+ 结构图 + 关键段落 + 你的解释
- 母题索引：你常遇到的母题及其不同处理方式
- 写作技法库：开篇、转场、伏笔、反讽、象征等的例子

## 写作/学习提示（针对本主题）
- 对一个解读：至少找 2 处文本证据（措辞/结构/意象）支撑
- 区分‘我喜欢’与‘文本有效’：分别写情感反应与形式理由
- 把你读过的作品按主题/结构分类，形成可检索的谱系
