# 实践与应用技能 (Praxis and Applied Skills)

这一部分用于把知识转化为可执行的方法：管理、教育、生产力、生活技能与工艺。目标是形成可复用的流程、清单与复盘机制。

## 这里关注什么
- 决策与复盘：目标、约束、风险、反馈与迭代
- 沟通与协作：写作、会议、知识传递与团队协作
- 个人系统：习惯、注意力、时间与精力管理（可验证）
- 生活技能：烹饪等可重复实践（以安全、可操作为准）

## 本目录建议沉淀的内容
- 流程模板：输入→步骤→验收标准→复盘问题→下一次改进
- 清单与标准：把隐性经验写成可执行条目
- 案例复盘库：做过的项目/任务按同一框架记录

## 写作/学习提示（针对本主题）
- 每个方法至少回答：适用场景、成本、失败模式、如何校准
- 避免空泛鸡汤：用可观察指标描述改进（例如每周输出、错误率）
- 把技能拆成最小练习单元并记录反馈回路
