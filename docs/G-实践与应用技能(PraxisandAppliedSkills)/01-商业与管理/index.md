# 商业与管理

商业与管理建议以‘价值创造—交付—反馈’组织：市场、产品、运营与组织管理如何协同，指标如何驱动行为。

## 这里关注什么
- 商业模型：价值主张、客户细分、渠道与成本结构
- 产品与运营：需求发现、实验、增长与留存（概念层）
- 组织与管理：目标管理、激励、沟通、招聘与文化
- 财务与风控：现金流、预算、合规与风险评估（概念层）

## 本目录建议沉淀的内容
- 案例拆解：一家产品/公司/策略的假设—执行—指标—结果—反事实
- 指标词典：北极星指标、漏斗指标、滞后/领先指标与误用点
- 管理工具箱：OKR、复盘、会议机制与决策记录模板

## 写作/学习提示（针对本主题）
- 写策略时：明确假设、可观测指标与失败阈值（何时止损/调整）
- 对组织问题：区分流程问题、激励问题与能力问题
- 每次复盘：写一个‘下次可执行的改动’，而不是泛泛总结
