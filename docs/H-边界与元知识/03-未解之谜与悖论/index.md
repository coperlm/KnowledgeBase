# 未解之谜与悖论

这里用于以严谨方式记录‘尚未闭合的问题’：把问题表述到可检验，区分概念悖论、经验未解与证据不足。

## 这里关注什么
- 经典悖论：逻辑/集合/概率/哲学悖论的结构与化解路径
- 科学未解：明确当前主流解释与证据缺口（按兴趣）
- 信息鉴别：高热度谜题常伴随误引与断章，需保留来源链

## 本目录建议沉淀的内容
- 问题卡：问题定义—已知事实—主流解释—替代解释—关键证据缺口
- 悖论拆解：前提集合、推理规则、冲突点与可能修正路径
- 来源记录：原始论文/数据/权威综述与二手传播的差异

## 写作/学习提示（针对本主题）
- 遇到‘谜题’：先列可验证事实，再列解释，不要反过来
- 写悖论：把它写成最短形式系统/命题集合，避免长叙事掩盖结构
- 记录你对证据的信心等级与原因（来源质量/可复现性）
