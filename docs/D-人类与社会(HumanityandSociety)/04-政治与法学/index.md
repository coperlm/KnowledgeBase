# 政治与法学

政治与法学的笔记最好按‘制度—激励—执行—争议’组织：法律文本只是起点，关键在执行结构、博弈与解释空间。

## 这里关注什么
- 政治制度：选举、分权、官僚体系、公共政策过程
- 法学基础：权利义务、法律推理、解释方法与判例体系（按需）
- 政治经济学：激励、租金、集体行动与公共选择
- 国际关系/比较政治（按兴趣）：权力、联盟、制度扩散

## 本目录建议沉淀的内容
- 概念框架：法治、正当程序、合法性、治理能力等的对照笔记
- 案例分析：一项制度/法律的目标—机制—执行—副作用—修正路径
- 争议议题的“钢人化”笔记：把不同立场论证写完整再评价

## 写作/学习提示（针对本主题）
- 写法律条文时：同时写它的适用边界与典型冲突场景
- 写政策评估时：先定义指标与反事实，再谈价值判断
- 对制度设计：记录激励相容性与可能被规避的路径
