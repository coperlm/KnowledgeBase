# 经济与金融

经济与金融适合沉淀‘模型—数据—机制’：用清晰假设解释行为与价格，用数据检验机制，而不是只记图表结论。

## 这里关注什么
- 微观：偏好、约束、均衡、博弈与机制设计直觉
- 宏观：增长、通胀、货币财政政策、商业周期
- 计量：因果识别、估计与偏误、稳健性（概念层）
- 金融：资产定价、风险、期限结构、市场微观结构（按需）

## 本目录建议沉淀的内容
- 模型假设清单：每个模型的关键假设与失效场景
- 常见数据指标速查：含义、口径、可比性与误读点
- 金融风险框架：波动、相关、尾部风险、杠杆与流动性（高层次）

## 写作/学习提示（针对本主题）
- 遇到结论：问‘识别来自哪里’（实验/自然实验/结构模型/相关性）
- 写金融概念：区分‘价格’与‘价值’以及对应信息假设
- 对任何策略：写风险暴露与失败条件，而不仅是收益叙事

## 边界（避免写到别处）
- 具体投资建议与买卖点（不在此处沉淀，避免误用）
