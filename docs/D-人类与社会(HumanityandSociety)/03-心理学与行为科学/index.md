# 心理学与行为科学

心理学与行为科学建议围绕‘可检验机制’整理：概念定义、测量工具、实验范式、统计推断与可重复性。

## 这里关注什么
- 认知与学习：注意、记忆、决策、强化学习与偏差
- 情绪与动机：情绪理论、压力、人格与自我调节
- 社会心理：从众、刻板印象、群体与身份、合作与冲突
- 方法：实验设计、量表、效应量、可重复性与开放科学实践（概念层）

## 本目录建议沉淀的内容
- 概念—测量对照：一个心理概念如何操作化、常见量表是什么
- 经典实验范式卡片：任务设计、关键操控、主要发现与争议
- 统计直觉笔记：p 值/置信区间/贝叶斯、功效与多重比较陷阱

## 写作/学习提示（针对本主题）
- 写研究结论时：记录样本、效应量与测量可靠性，而不只写显著性
- 遇到‘偏差’条目：写它出现的条件与减少它的策略（高层次）
- 对心理概念：写‘可观察预测’而不是只写解释性故事
