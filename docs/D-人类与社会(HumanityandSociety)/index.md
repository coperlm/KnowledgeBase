# 人类与社会 (Humanity and Society)

这一部分用于沉淀关于人类行为、制度与文化的分析框架：事件与制度如何形成，个体与群体如何互动，研究方法如何支撑结论。

## 这里关注什么
- 历史视角：事件链、结构性因素、史料与解释框架
- 社会科学方法：变量、因果识别、调查与实验、解释与预测
- 制度与权力：法律、政治经济结构、激励与约束
- 心理与行为：决策、偏差、社会认同与群体动力学

## 本目录建议沉淀的内容
- 概念框架卡片：用 1 页解释一个理论（定义—机制—预测—证据—批评）
- 案例库：把具体事件按‘背景—机制—结果—反事实’记录
- 常用数据与方法备忘：问卷、访谈、文本分析、计量与实验设计

## 写作/学习提示（针对本主题）
- 写社会科学结论时：分清‘描述性事实’与‘规范性判断’
- 对因果主张：记录识别策略（自然实验/工具变量/差分等，哪怕只是概念层）
- 把你的价值前提显式化，避免把立场当事实
