# 社会与人类学

社会与人类学适合整理‘解释框架与田野视角’：制度、规范与意义如何塑造行为，以及研究者如何通过方法获得证据。

## 这里关注什么
- 社会结构：阶层、网络、组织与制度安排
- 文化与意义：符号、仪式、身份、叙事与规范
- 方法：田野调查、访谈、参与式观察与伦理问题
- 宏观与微观连接：结构如何约束个体，个体如何改变结构

## 本目录建议沉淀的内容
- 关键概念卡：结构/能动、文化相对主义、他者化、社会建构等
- 案例笔记：一个社区/组织的规则、冲突、资源与象征体系
- 方法论备忘：抽样、偏差、研究者位置性与可复核性

## 写作/学习提示（针对本主题）
- 写观察时：区分‘观察到的行为’与‘参与者的解释’与‘你的解释’
- 对制度：写激励约束与非正式规范的互动，而不只写条文
- 对文化解释：列替代解释，避免把单一叙事当万能钥匙
